## 📋 项目概述

这是一个混合使用 **C++**、**Python** 和 **Rust** 的演示项目，展示了如何在一个项目中无缝集成多种编程语言的优势。

## ✨ 主要特性

- 🔄 多语言互操作能力
- 🛡️ 类型安全的接口设计
- 🧩 模块化架构
- 🚅 高性能计算支持

## 🛠️ 技术栈

| 语言 | 用途 | 版本要求 |
|------|------|----------|
| **C++** | 用于高性能计算组件 | C++23 或更高 |
| **Python** | 用于数据处理和脚本 | Python 3.12+ |
| **Rust** | 用于安全和并发部分 | Rust 1.85+ |

## 🔧 安装与设置

```shell
# 克隆仓库
git clone https://github.com/WangWindow/xxx.git
cd xxx

# 安装依赖
uv sync
```
